AVISO DE PRIVACIDAD

Política de privacidad

La aplicación CuestionarioApp desarrollada por Andy Nelson Ramos Villalobos recolecta datos como son: un correo electrónico, una contraseña de acceso para la aplicación, nombre, apellido, y tipo de usuario (estudiante o docente) al realizar el registro de una cuenta de usuario, esto para ofrecer una interfaz adecuada al usuario que estará usando la aplicación, ya que, existen dos escenarios, uno para el estudiante y otro para el docente mismos que se eligen al momento del registro.

Cabe resaltar que, el nombre del usuario es utilizado para identificar los documentos que la aplicación Android genera como evidencia luego de que el usuario finalice su cuestionario.

Por otro lado, se recolecta información acerca del dispositivo Android en el que se instala la aplicación; tales como: ID, marca y modelo. Esto sucede una vez que el usuario inicia la “PRUEBA GRATUITA” y se utiliza para llevar el control del tiempo de prueba ofrecido a cada usuario en un determinado dispositivo Android.

Toda la información que se captura se almacena en las hojas de cálculo de la cuenta de la aplicación Android disponible en el servidor de google (Google Drive) y son tratadas de manera confidencial sin acceso a terceros.
